# siteweb-i-guide

## 项目简介

`siteweb-i-guide` 是一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定的域名或网站，而是收集整理各种独立的网页资源，以便于浏览、参考和分享。

这些页面可能包含技术文档、工具界面、知识卡片、指南手册或其它形式的静态内容。每个 HTML 文件均为自包含的独立页面，无需依赖外部服务器或动态运行环境。

## 目录结构

```
siteweb-i-guide/
├── pages/          # 存放所有独立的 HTML 页面
│   ├── example-page.html
│   └── ...
├── assets/         # 页面所需的静态资源（图片、样式、脚本等）
│   ├── images/
│   ├── styles/
│   └── scripts/
├── docs/           # 相关文档或说明文件
├── index.html      # 仓库首页（可选），用于导航或展示页面列表
├── README.md       # 本文件
└── LICENSE         # 许可协议
```

## 页面归档说明

- 每个 HTML 页面应尽量保持独立，避免外部资源依赖（若有必要，可将资源放入 `assets/` 目录并引用相对路径）。
- 页面命名应简洁、有意义，建议使用英文小写字母、数字和连字符。
- 若页面包含敏感信息或动态内容，请勿提交至本仓库。
- 欢迎提交新的独立页面，但需确保内容不违反任何法律法规或平台政策。

## 如何贡献

1. Fork 本仓库。
2. 创建新的分支：`git checkout -b add-your-page`。
3. 将 HTML 文件放入 `pages/` 目录，必要时更新 `assets/` 和 `docs/`。
4. 提交并推送分支，然后发起 Pull Request。
5. 维护者审核后合并。

- 页面内容可能随时间更新或移除，恕不另行通知。
- 如发现页面无法正常显示或包含错误，欢迎提交 Issue 或 Pull Request。
